# Codeforces Beginner Practice Sheet (Rating < 1200)

A mixed-topic problem set to build core fundamentals: implementation, basic math, simple greedy, arrays/strings, and intro ad-hoc logic. Solve in order; don't skip around too much at first.

## How to use this sheet
- Aim for 30–45 minutes per problem before checking editorial/hints.
- Solve in your main language, then read the editorial even if you got it right (shorter/cleaner solutions exist).
- Re-attempt any problem you couldn't solve, 1 week later, from scratch.
